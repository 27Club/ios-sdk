SPTTrackPlayer Class Reference

Inherits from NSObject
Declared in SPTTrackPlayer.h

Overview

This class ties together SPTAudioStreamingController and SPTCoreAudioController to provide a basic audio player. For more advanced playback setups, use the aforementioned classes directly.

skipToPreviousTrack:

Skips playback to the previous track.

- (void)skipToPreviousTrack:(BOOL)disallowSameTrackRestart

Parameters

disallowSameTrackRestart

If set to YES, playback will always be skipped to the track prior to the current one, bypassing default behaviour to restart the current track.

Discussion

The default behaviour of this method will restart the current track if playback is sufficiently progressed, matching the behaviour of most other media players.
